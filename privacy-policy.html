隐私政策

生效日期： 2024年1月1日

感谢您使用“HuleSheetTools”（以下简称“本插件”）。本插件是作为 Google Docs™（谷歌文档）的附加组件（Add-on）提供的。我们将通过本隐私政策向您说明本插件如何收集、使用、存储和保护您的信息，特别是涉及 Google 用户数据的部分。

1. 我们收集和访问的信息

为了实现插件的功能（例如：文档处理、格式化、内容生成等），我们需要访问您的 Google 账号下的部分数据。具体包括：

  • Google 文档内容： 当您在文档中运行本插件时，我们可能会读取您当前打开的文档内容，以便执行您请求的操作（如查找替换、翻译、格式调整等）。
  • 基本账号信息： 我们可能会获取您的电子邮件地址或用户 ID，用于识别用户身份及保存您的插件设置偏好。
  • 运行日志： 为了排查故障，我们可能会收集插件运行时的错误日志（不包含文档具体内容）。

2. 数据使用方式（Google API 服务用户数据政策）

本插件对从 Google API 接收的信息的使用和传输到任何其他应用程序，将严格遵守 Google API 服务用户数据政策，包括“限制使用”（Limited Use）要求。

  • 仅用于提供功能： 我们仅在提供面向用户的功能所必需时，才会使用您的数据。
  • 不用于广告： 我们绝不会使用您的 Google Workspace 数据来向您展示广告。
  • 不通过人工读取： 除非为了安全目的（如调查滥用行为）、遵守法律法规或经您明确同意，否则我们要么不读取您的数据，要么仅通过自动化代码处理数据。

3. 数据存储与保留

本插件主要在您的浏览器本地或 Google 的服务器端运行。我们通常不会将您的文档内容传输到我们的服务器进行持久化存储。如果某些功能需要临时传输数据（例如调用第三方翻译API），数据将在处理完成后立即删除。

4. 第三方服务

如果本插件集成了第三方服务（如 OpenAI、翻译服务等），在使用这些特定功能前，我们会明确提示您。这些第三方服务只能根据我们的指示处理数据，且受保密协议约束。

5. 数据安全

我们采取合理的安全措施来保护您的信息，防止未经授权的访问、使用或泄露。所有的网络通信均通过 HTTPS 加密协议传输。

6. 变更通知

我们可能会不时更新本隐私政策。更新后的政策将发布在本页面上。继续使用本插件即表示您同意更新后的隐私政策。
